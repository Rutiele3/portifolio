Portfólio

Sistema de Agendamento de Consultas

O sistema foi desenvolvido em Python, usando programação orientada a objetos (POO) para organizar classes e funcionalidades. A interface gráfica foi feita com a biblioteca CustomTkinter, que oferece design moderno e componentes personalizáveis. Para mensagens amigáveis de sucesso, erro e confirmação, utilizou-se a CTkMessagebox. O sistema aplica conceitos de abstração (com classes abstratas) e encapsulamento para proteger dados sensíveis, como senhas e preços.
